# LINEアカウント管理サイト（作成中）

## 使用技術

- **フロントエンド**
  - React
  - TypeScript
  - shadcn/ui
- **バックエンド**
  - Django (Django REST Frameworkを使用)
- **デザイン**
  - Tailwind CSS
- **データベース**
  - SQLite
